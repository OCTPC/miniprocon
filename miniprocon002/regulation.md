# 切り餅発見伝
切り餅が好きなので，切り餅(四角形)を探してください．


## 入力
入力は以下の形式で与えられる．
```
N M
line_1
line_2
:
line_N
```
1行目のN, Mは格子の大きさである．ただし，9 <= N * M <= 10^6である．  
2行目以降はN行*M文字の( #|. )が与えられる．  

## 出力
切り餅の総数を出力せよ．切り餅とは辺がすべて#である長方形であり，辺の長さは3文字以上である．

## 例

### 入力例0
```
3 3
###
#.#
###
```
### 出力例0
1

### 入力例1
```
5 10
...###.###
...#####.#
...###.###
.....###..
..........
```
### 出力例1
3

### 入力例2
```
10 15
#.######.#.###.
#..#...########
..######...#.##
.#.###...#.#..#
.###.######.#..
######..#######
###..##.#.#.#.#
#.#############
#.#.##..#.###.#
####..####.#.#.
```
### 出力例2
16

## 勝敗条件
最も経過時間の短かった回答を提出した人が勝利となります．
